# 医療広告コンプライアンスチェッカー

整骨院・鍼灸院・整体院などのコンテンツが、医療広告ガイドライン・あはき法・柔整法に違反していないかをチェックする。

## クイックスタート

1. 対象コンテンツを読み込む
2. 下記チェックリストに沿って確認
3. 結果をチェックリスト形式で出力

---

## チェックリスト

### 1. 効果・効能の断定表現

| NG表現 | OK表現 |
|--------|--------|
| 治る、治す、完治 | 改善が期待できる、緩和を目指す |
| ○○に効く、効果がある | ○○のケアに取り組む、○○にアプローチ |
| 必ず良くなる、絶対に | 〜につながることがあります |
| 痛みが取れる | 痛みの軽減を目指す |

**チェック項目**:
- [ ] 断定的な治療効果の表現がないか
- [ ] 効能を保証する表現がないか

### 2. 最上級・比較表現

| NG表現 | OK表現 |
|--------|--------|
| 最高、No.1、日本一 | （具体的データ＋出典がある場合のみ可） |
| 地域で一番、他院より優れた | 当院の特徴として〜 |
| 唯一の、ここだけの | 〜を重視した施術 |

**チェック項目**:
- [ ] 最上級表現がないか
- [ ] 他院との比較表現がないか

### 3. 医療行為の暗示（柔整・あはき共通）

| NG表現 | OK表現 |
|--------|--------|
| 診断、治療、治療行為 | 評価、施術、ケア |
| 処方、投薬 | （使用不可） |
| 医学的、医療的 | 専門的なアプローチ |

**チェック項目**:
- [ ] 医師法に抵触する表現がないか
- [ ] 医療行為を暗示する表現がないか

### 4. あはき法第7条（鍼灸・あん摩マッサージ指圧）

**広告可能事項（これ以外は原則NG）**:
- 施術者の氏名・住所
- 業務の種類（はり、きゅう、あん摩、マッサージ、指圧）
- 施術所の名称・電話番号・所在地
- 施術日・施術時間
- 予約の有無
- 出張による施術の有無
- 駐車場の有無

**広告禁止事項**:
- [ ] 施術者の経歴・学歴の記載がないか
- [ ] 施術方法・テクニック名の詳細記載がないか
- [ ] 料金の記載がないか（ウェブサイト除く）
- [ ] 効能効果の記載がないか

### 5. 柔整法第24条（柔道整復師）

**広告可能事項（これ以外は原則NG）**:
- 柔道整復師である旨
- 施術者の氏名・住所
- 施術所の名称・電話番号・所在地
- 施術日・施術時間
- 予約制の有無
- 出張施術の実施
- 駐車場の有無
- 健康保険の取り扱い

**広告禁止事項**:
- [ ] 施術者の経歴・技術の優位性の記載がないか
- [ ] 適応症・効能効果の記載がないか
- [ ] 「○○専門」などの専門性の強調がないか

### 6. 体験談・ビフォーアフター

**チェック項目**:
- [ ] 体験談に「個人の感想です」等の注記があるか
- [ ] ビフォーアフター写真に「個人差があります」の注記があるか
- [ ] 結果を保証するような表現がないか

### 7. 誘引性のある疾患名の使用

**チェック項目**:
- [ ] 疾患名で集客を誘引する表現がないか
- [ ] 「○○でお悩みの方へ」が過度に強調されていないか

|----------|------|----------|

| 該当箇所 | 問題点 | 修正案 |

---

## 判定基準

| 判定 | 基準 |
|------|------|
| ⭕ 問題なし | 違反表現なし、そのまま公開可能 |
| ⚠️ 要修正 | 軽微な問題あり、修正後に公開可能 |
| ❌ 公開不可 | 重大な違反あり、大幅な修正が必要 |
