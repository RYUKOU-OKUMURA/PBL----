# LINEコラム執筆スキル

整体院・治療院のLINE配信用コラムを、一貫した品質で執筆するためのガイド。

## 文体・トーン

### 基本ルール
- **敬体**: です・ます調
- **トーン**: 温かく親しみやすい、励まし
- **感嘆符**: 強調したい箇所に「！」を戦略的に使用
- **専門用語**: 使わない（どうしても必要なら直後に平易説明）

### 絵文字ルール
| 絵文字 | 用途 |
|-------|------|
| 💦 | 心配・困りごとの共感 |
| 🌸 | ポジティブな話題・季節感 |
| ☀️ | 明るい話題・夏の季節感 |
| 🍂 | 秋の季節感 |
| ❄️ | 冬の季節感・冷えの話題 |
| (^^)/ | 締めの挨拶（必須） |

**注意**: 絵文字は控えめに。1記事に2〜3個程度。

### 個人的なタッチ
家族のエピソードや日常の小話を適宜入れる。

```
うちの娘も最近よく水を飲み忘れるので、
一緒にコップ1杯チャレンジしています！
```

## 記事構成（5パート）

### 1. 季節・天候フック（1〜2文）
季節感や天候から入り、読者の「今」に寄り添う。

```
急に寒くなりましたね💦
朝晩の冷え込みが厳しくなってきました。
```

```
梅雨入りしましたね☔
ジメジメした日が続いています。
```

### 2. 問題提起（2〜3段落）
読者が「あ、自分のことだ」と感じる悩みを提示。

**ポイント:**
- 「〜していませんか？」と問いかける
- 具体的な場面を描写
- 共感を示す

```
こんな時期、なんだか体がだるい…
肩がいつもより重い…
そんな風に感じていませんか？

実は、季節の変わり目は体調を崩しやすい時期なんです。
気温差で自律神経が乱れやすくなるんですね。
```

### 3. 専門的洞察（1〜2段落）
なぜそうなるのか、シンプルに説明。

**ポイント:**
- 専門用語は避ける
- 具体的な数字を1つ入れる（「頭の重さは約5kg」など）
- 比喩でわかりやすく

```
私たちの頭は約5kgもあるんです。
ボウリングの球くらいの重さですね！

それを首と肩で支えているので、
姿勢が崩れると負担が何倍にもなってしまいます。
```

### 4. 実践アドバイス（1〜2段落）
今日からできる具体的なケア方法。

**ポイント:**
- 1つに絞る（多くても2つ）
- 具体的な量・回数を示す
- ハードルを下げる表現

```
まずは朝起きたら、コップ1杯の白湯を飲んでみてください。
たったこれだけで、体の巡りがぐっと良くなりますよ🌸

お風呂上がりに首をゆっくり回すのもおすすめです！
左右5回ずつ、痛くない範囲でOKです。
```

### 5. 励ましの締め（1〜2文）
前向きなメッセージと(^^)/で締める。

```
小さなケアの積み重ねが、
90歳でも自分の脚で歩ける体をつくります。
今日もお体大切に(^^)/
```

```
無理せず、できることから始めてみてくださいね。
いつでも相談お待ちしています(^^)/
```

## 長さ・フォーマット

### 文字数・行数
- **目安**: 15〜25行
- **1行**: 全角20〜25文字程度
- **段落**: 2〜4行で空行を入れる

### モバイルファースト
- スマホで読みやすい短い段落
- 長文は避ける
- 箇条書きは基本使わない

## コアメッセージの活用

以下のメッセージを自然に織り込む（毎回ではなく、適宜）：

| メッセージ | 使用頻度目安 |
|-----------|-------------|
| 水分補給の大切さ | 月2〜3回 |
| ストレッチの習慣化 | 月2回 |
| 「90歳でも自分の脚で歩ける体」 | 年6回 |

## 具体的な数字・表現

読者の記憶に残る具体的な表現を使う：

| 抽象的 | 具体的 |
|-------|-------|
| 水を飲む | コップ1杯の白湯 |
| 頭は重い | 頭の重さは約5kg（ボウリングの球） |
| こまめに動く | 1時間に1回、席を立つ |
| ストレッチする | 首を左右5回ずつ回す |
| 姿勢に気をつける | 耳と肩が一直線になるように |

## 避けるべき表現

- 医学専門用語
- 長い説明文
- 脅かす表現（「〜しないと大変なことに！」）
- 複雑なケア方法（3ステップ以上）
- 効果の断定（「必ず治る」「絶対に」）

## 季節別テーマ例

| 季節 | テーマ例 |
|-----|---------|
| 春 | 花粉症と肩こり、新生活の疲れ、寒暖差 |
| 梅雨 | 湿気とだるさ、むくみ、気圧の変化 |
| 夏 | 冷房による冷え、水分補給、夏バテ |
| 秋 | 寒暖差、ぎっくり腰、乾燥 |
| 冬 | 冷え、血行不良、年末年始の疲れ |

## サンプル記事

```
急に寒くなりましたね💦

朝晩の冷え込みで、
体がギュッと縮こまっていませんか？

実は、寒くなると私たちの体は
無意識に肩をすくめてしまうんです。

これが続くと、肩や首がガチガチに…
なんだか頭も重く感じますよね。

そんな時は、お風呂でゆっくり温まった後、
首をゆーっくり回してみてください🌸

左右5回ずつ、痛くない範囲でOKです！

小さなケアの積み重ねが、
元気な体をつくっていきます。

寒い日が続きますが、
お体大切にお過ごしくださいね(^^)/
```
