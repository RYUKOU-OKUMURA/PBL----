# Slidev PDFエクスポート手順

## 推奨手順

### 1. 依存関係のインストール
```bash
npm install
```

### 2. 開発サーバーの起動（確認用）
```bash
npm run dev
```

### 3. PDFエクスポート
```bash
npm run export-pdf
```

## マーメイド図が表示されない場合の対処法

### オプション1: プレゼンテーションモードでエクスポート
```bash
slidev export 思春期特発性側弯症_slidev.md --format pdf --output presentation.pdf --wait 5000
```

### オプション2: 手動での画像生成
1. ブラウザでSlidevを開く
2. 各マーメイド図を個別にスクリーンショット
3. 画像として保存し、スライドに挿入

### オプション3: HTML出力を利用
```bash
slidev build 思春期特発性側弯症_slidev.md
```
その後、ブラウザでHTMLを開いてPDF印刷

## トラブルシューティング

### 問題: マーメイド図が空白で表示される
**解決法:**
- `--wait 5000` オプションを追加して読み込み時間を確保
- Playwrightブラウザーを最新版に更新

### 問題: フォントが正しく表示されない
**解決法:**
- システムフォントの確認
- フォールバックフォントの設定

### 問題: ファイルサイズが大きすぎる
**解決法:**
- 画像の最適化
- マーメイド図のサイズ調整
