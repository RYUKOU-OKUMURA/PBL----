## 実践的な編集テクニック

### 1. 薬機法・医療広告ガイドライン違反のチェック
```
NG表現：
- 「○○が治る」「完全に治す」
- 「○○に効く」「○○に効果がある」（断定的）
- 「○○を防ぐ」（断定的）

OK表現：
- 「○○が改善する可能性がある」
- 「○○に役立つ可能性がある」
- 「○○を防ぐ一助になる」
- 「○○が緩和されることが期待されます」
```

### 2. 文体の統一
```
統一すべき表現：
- 「です・ます調」で統一
- 「〜です。」「〜ます。」で終わる
- 「〜だ。」「〜である。」は使わない

親しみやすさと専門性のバランス：
- 専門用語を使う時は必ず説明を入れる
- 「○○とは、簡単に言うと○○のことです。」
- 患者さんのエピソードを入れると親しみやすくなる
```

### 3. 文章の流れを良くする
```
改善ポイント：
- 一文が長すぎる場合は2文に分ける（60文字以内を目安）
- 段落の最初に要点を書く
- 具体例の前に「例えば、」を入れる
- 読者の疑問に答える順序で書く
```

### 4. SEO対策
```
チェックポイント：
- タイトルにSEOキーワードが含まれているか
- 最初の段落（導入）にSEOキーワードが含まれているか
- 見出し（H2）にSEOキーワードが含まれているか
- 関連キーワードも自然に含まれているか
- キーワードの詰め込みすぎは避ける（自然に）
```

## よくある問題と対処法

### 1. 情報の正確性に疑問がある場合
```
対処：
1. Worker1に再リサーチを依頼
2. 信頼できるソース（医学論文、公的機関）を確認
3. 根拠が不明確な情報は削除または修正
```

### 2. 文体が統一されていない場合
```
対処：
- です・ます調に統一
- 「だ・である調」を「です・ます調」に修正
- 敬語の使い方を統一
```

### 3. 専門用語が説明されていない場合
```
対処：
- 専門用語の後に説明を追加
- 「○○とは、簡単に言うと○○のことです。」
- または括弧で説明を追加
```

### 4. 来院への導線が不自然な場合
```
対処：
- 営業感が強すぎる場合は削除または修正
- 自然な流れで来院への誘導を含める
- 「もし、○○が気になる場合は、無理をせず、早めに専門家に相談することをおすすめします。」
```

## 成功のための実践原則

1. **継続的な品質管理**
   - 各段階でしっかりチェック
   - 問題は早期に発見・修正

2. **明確な指示**
   - Workerへの指示は具体的に
   - 成功基準を明確に

3. **効率的なワークフロー**
   - Worker1→Worker2→Worker3の順序を守る
   - 各段階で成果物を確認してから次へ

4. **読者目線での編集**
   - 読者が理解しやすいか
   - 読者が価値を感じられるか
