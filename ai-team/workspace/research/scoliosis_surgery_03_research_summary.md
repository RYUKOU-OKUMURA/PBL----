# 脊柱側弯症の手術後リハビリに関するリサーチサマリー（第3回）

## テーマ
脊柱側弯症の手術後のリハビリテーションと生活

---

## 1. 主要な情報の要点まとめ

### 1.1 術後リハビリの段階的スケジュール

| 期間 | 内容 | 目安 |
|------|------|------|
| **術後1〜2日** | 安臥、疼痛管理開始、呼吸練習 | ベッド上安静 |
| **術後2〜3日** | 理学療法士の指導下で歩行開始、立ち上がり練習 | 歩行器〜歩行 |
| **術後5〜7日** | 歩行安定、日常生活動作練習、階段昇降 | 独歩可能 |
| **術後1〜2週** | 退院 | 約10日〜2週間の入院 |
| **退院〜3ヶ月** | 軽い運動、歩行、ストレッチ | デスクワーク可能 |
| **3〜6ヶ月** | 軽いジョギング、非接触スポーツ可能 | 学校・仕事復帰 |
| **6ヶ月以降** | 接触スポーツ、本格的スポーツ活動可能 | 運動解禁 |

### 1.2 コルセット装着期間

- **期間**: 術後3〜6ヶ月
- **目的**: 脊椎の安定化、骨癒合の促進
- **24時間装着**: 初期は24時間、徐々に装着時間短縮

### 1.3 院内リハビリプログラム

| 内容 | 詳細 |
|------|------|
| **呼吸訓練** | 肺活量改善、深呼吸練習（術後の肺機能低下対策） |
| **筋力トレーニング** | 下肢ストレッチ、体幹筋力トレーニング（軽いものから開始） |
| **バランス訓練** | 片足立ち、バランス感覚の回復（手術で変化したバランスへの適応） |
| **日常生活動作練習** | 洗顔、着替え、移乗動作 |

## 3. 術後生活の実態

### 3.1 学校・仕事復帰の時期

| 期間 | 復帰可否・目安 |
|------|--------------|
| **2〜4週間** | デスクワーク、座位での学習可能（場合による） |
| **8〜12週（2〜3ヶ月）** | 一般的な学校・仕事復帰の目安 |
| **3ヶ月以降** | ランニング・軽い運動可能 |
| **6ヶ月** | 非接触・接触スポーツ可能 |

### 3.2 術後の生活制限

| 制限項目 | 期間・内容 |
|---------|----------|
| **前かがみになる動作** | 3〜6ヶ月まで制限 |
| **体をひねる動作** | 3〜6ヶ月まで制限 |
| **重いものを持ち上げる** | 3〜6ヶ月まで禁止（10kg以上は6ヶ月まで禁止） |
| **スポーツ活動** | 6ヶ月間は禁止 |
| **体育・授業** | 6ヶ月間は見学または制限 |

### 3.3 長期的な制限

- 激しい衝突を伴うスポーツ（ラグビー、アメリカンフットボール等）は個別判断
- 重いものを扱う職業・活動は医師と相談
- 極度の脊柱伸展・屈曲を伴う動作は慎重に

## 4. 妊娠・出産への影響

- 術後妊娠・分娩は可能
- 骨盤位（臀位）の場合は帝王切開を選択することもある
- 経膣分娩・帝王切開の選択は側弯の程度、骨盤の形態、全身状態等を総合的に判断
- **注意**: 妊娠を希望される場合は、手術担当医および産婦人科医への事前相談が必要

## 8. 当院ブログで差別化できるポイント

### 8.1 整体院としての独自視点

- 手術で背骨が真っすぐになった後も、筋肉のバランスが不揃いで痛みが残ることがある
- 整体院ができる術後のアライメント調整を提案
- 術後のボディバランス改善へのアプローチ

### 8.2 未開拓の切り口

| 差別化ポイント | 説明 |
|---------------|------|
| **成人側弯症手術後のケア** | 中高年での手術例や、変性側弯症術後のケア |
| **術後10年以上の長期的ケア** | 手術から年月が経過してからの腰痛や肩こりへのアプローチ |
| **家族のサポート方法** | 手術を受ける本人だけでなく、家族がどう支えるべきか |
| **手術「前」の準備としての整体** | 手術を控えている人への術前セルフケア指導 |
| **具体的なセルフエクササイズ** | 自宅でできる安全なストレッチやエクササイズ |

### 8.3 推奨される記事構成案

```
# 側弯症手術後、整体でできること【整体師が解説】

## はじめに
- 手術で背骨が真っすぐになっても、「まだなんか違和感がある」という声
- 実際に当院に来られた手術後の患者さんのエピソード

## 手術で変わること・変わらないこと
- 背骨の形は変わるが、筋肉の使い方はまだ手術前の癖が残っている
- バランス感覚が変わることで生じる新たな不安定さ

## 術後に整体でできる3つのアプローチ
1. 筋肉のバランス調整
2. 姿勢の再教育
3. 呼吸と肋骨の可動域改善

## 注意点（絶対にしてはいけないこと）
- 医師の許可なく整体を受けるのはNG
- 術後の時期ごとの注意点

## 具体的なセルフケア
- 自宅でできる安全なストレッチ
- 姿勢チェックの方法

## まとめ
- 手術はゴールではなくスタート
- 無理のない範囲で整体と上手に付き合う方法
```

## 11. 最新エビデンス（2025年時点）

### 11.1 長期QOL（10年以上）

- 10年後SRS-22スコア：平均4.5/5.0（非常に良好）
- 10年後ODI：平均5.6%（最小の障害）
- 40年後QOL：一般人と同等
- 2年後低QOL患者の多くが5年後には改善

### 11.2 スポーツ復帰

- 6ヶ月で50%以上が接触・非接触スポーツに復帰
- 79%が接触スポーツで術前レベルに復帰
- 復帰による合併症は稀

### 11.3 術後疼痛管理

- ケタミン併用：モルヒネ消費量減少、疼痛改善
- ESPブロック：術後1-3日目の疼痛軽減
- ポピドンヨード灌流：手術部位感染70%減少
