# 脊柱側弯症の手術方法に関するリサーチサマリー（第2回）

## テーマ
脊柱側弯症の最新の手術方法と技術

## 1. 主要な情報の要点まとめ

### 1.1 手術方法の種類と特徴

| 手術法 | 特徴 | 適応 |
|--------|------|------|
| **後方矯正固定術（PSF）** | 最も一般的、確立された術式 | 重症例、長いカーブ、二重カーブ、胸椎カーブ |
| **前方矯正固定術** | 可動椎間を多く残せる、固定範囲を小さくできる | 胸腰椎カーブ、腰椎カーブ |
| **前方後方同時アプローチ** | LIF併用、変性腰椎側弯症で出血が少ない | 変性側弯症 |
| **最小侵襲手術（MIS）** | 出血少、回復早、切開小 | 軽度〜中度の側弯 |
| **ロボット支援手術** | ねじ精度向上、合併症減少 | 複雑な変形 |
| **椎体タンザリング固定（VBT）** | 非融合、成長を維持 | 若年者、成長余力のある症例 |

### 1.2 後方矯正固定術（PSF）の詳細

**手術手技**
- 背部正中に切開
- すべての椎体にスクリュー（ pedicle screw ）で固定
- ロッドで矯正・固定
- 自己骨または人工骨を移植して骨癒合を促進

**歴史的背景**
- 1986年：Steffeeが椎弓根スクリュー法を普及
- 1990年代：CD法（Cotrel-Dubousset法）、TSRH、ISOLA等が日本に導入
- 現在：全椎体スクリュー固定が標準

### 1.3 最小侵襲手術（MIS）の詳細

**特徴**
- 小切開（5-12cm程度）
- 筋肉を温存（正中脊柱筋を切開せず）
- 出血量が少ない
- 術後疼痛が軽減
- 早期退院可能

**導入施設（日本）**
- 成田赤十字病院：MISt（ミスト）
- 東京慈恵会医科大学：低侵襲手術
- 岐阜大学：10-12cm切開
- 順天堂大学：5mm程度小切開
- 高知大学、兵庫医科大学：MISt実施

### 1.4 ロボット支援手術の詳細

**Mazor X ロボットシステム導入施設（日本）**
| 施設 | 導入時期 | 備考 |
|------|----------|------|
| 聖マリアンナ医科大学病院 | 2021年4月 | 日本初 |
| 牧田病院（東京） | 2024年5月 | 東京初 |
| 八尾徳洲会総合病院 | - | 順調に稼働 |
| 千葉県内の施設 | 2025年4月予定 | 千葉県初 |

**メリット**
- 椎弓根スクリューの設置精度向上
- 術中合併症の減少
- 正確な手術計画の実行

**注意点**
- 重度側弯症では使用困難な場合あり
- 精度不足（73.6%）、ソフトウェアエラー（13.7%）の報告あり

## 2. 引用可能なデータ・統計

### 2.1 手術時間・出血量・入院期間

| 項目 | 従来法（PSF） | 最小侵襲（MIS） | 出典 |
|------|-------------|---------------|------|
| **手術時間** | 3-7時間（平均248-340分） | 短縮傾向 | 日本のデータ・PubMed |
| **出血量** | 320-1,260ml | 64.2ml | 日本のデータ・PubMed |
| **入院期間** | 2週間（9.3日） | 1週以内で退院可能 | 日本のデータ・PubMed |
| **輸血率** | - | TXA使用で低下 | 2024年メタアナリシス |

### 2.2 矯正率

| カーブ部位 | 平均矯正率 | 出典 |
|-----------|-----------|------|
| **主胸椎（MT）カーブ** | 81%（範囲：64-98%） | 北海大学研究 |
| **胸腰椎・腰椎（TL/L）カーブ** | 75%（範囲：36-98%） | 北海大学研究 |
| **全体** | 60-80% | 複数の報告 |

### 2.3 合併症率

| 合併症 | 率 | 出典 |
|--------|-----|------|
| **成人脊柱変形手術（全体）** | 11.8% | 全国調査 |
| **75歳以上** | 有意に高値 | 全国調査 |
| **ロボット支援（術中）** | 4.1% | PRoGRSS中間解析 |
| **ロボット支援（術後）** | 6.6% | PRoGRSS中間解析 |
| **再手術率** | 1.5% | PRoGRSS中間解析 |
| **PJK率（1年後）** | 5.6% | 2025年研究 |

### 2.4 主な合併症の種類

- 神経損傷
- 感染症
- 深部静脈血栓症
- 偽関節（骨が癒合しない）
- 近位部接合部後弯（PJK）
- スクリューの緩み・破損

### 2.5 手術成績に関するデータ

| 項目 | 数値 | 出典 |
|------|------|------|
| 神経筋性側弯症手術の満足度 | 92% | 日本の研究 |
| TXAによる出血減少量 | 平均 -681.81ml | 日本の研究 |
| MISの矯正率 | PSFと同等 | 2024年研究（84例） |

## 3. 最新の手術技術・器材

### 3.1 主なインプラントシステム

| システム名 | 説明 |
|-----------|------|
| **CD法** | Cotrel-Dubousset法、3次元矯正が可能 |
| **TSRH** | Texas Scottish Rite Hospital instrumentation |
| **ISOLA** | 多目的な脊椎固定システム |
| **Pedicle Screw法** | 椎弓根スクリュー固定、現在の標準 |

### 3.2 周術期管理の進歩

**トロキサム酸（TXA）**
- 高用量TXAは低用量より出血量・輸血量を有意に減少
- 安全性は両用量で同等

**ERASプロトコル（回復促進）**
- 入院期間短縮に寄与
- 合併症減少傾向
- 患者満足度向上

## 4. 術後の経過と回復

### 4.1 典型的な術後スケジュール

| 期間 | 内容 |
|------|------|
| **手術日** | ICU入室、疼痛管理 |
| **術後1-2日** | ベッド上安静 |
| **術後3-7日** | 歩行開始、装具装着 |
| **術後1-2週間** | 退院 |
| **術後1ヶ月** | 外来受診、軽微な運動開始 |
| **術後2-3ヶ月** | 通常の学校・仕事復帰 |
| **術後5-6ヶ月** | 装具終了（骨癒合確認後） |

### 4.2 術後の制限と注意点

- 半年〜1年間の激しい運動禁止
- 接触スポーツは6ヶ月以上禁止
- スクリュー周囲の疼痛（年齢による）
- 定期的なレントゲン検査
